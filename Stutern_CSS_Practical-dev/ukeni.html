Paddings, Margins and Borders explained

Paddings: The CSS padding properties are used to generate space around an element's content, inside of any defined borders. With CSS, you have full control over the padding. There are properties for setting the padding for each side of an element (top, right, bottom, and left).

Note:Negative values are not allowed.

Margins: The CSS margin properties are used to create space around elements, outside of any defined borders. With CSS, you have full control over the margins. There are properties for setting the margin for each side of an element (top, right, bottom, and left).

Note:Negative values are allowed.

Borders The CSS Border properties are used to generate space between the paddings and margins. It allows you to specify the style, width, and color of an element's border.
